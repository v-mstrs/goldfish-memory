# Goldfish Memory

A browser extension that helps you remember characters in web novels by highlighting their names and showing tooltips with descriptions and images. Supports automatic backups and site-specific configurations.

## Requirements

- **Node.js**: Version 18 or higher (v20+ recommended).
- **npm**: Installed automatically with Node.js.
- **Browser**: Google Chrome, Edge, Brave, or Mozilla Firefox.

## Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd goldfish-memory
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

## Building the Extension

To build the extension for production (creates `dist-chrome` and `dist-firefox` folders):

```bash
npm run build
```

## How to Run

### Google Chrome / Edge / Brave

1.  Open your browser and navigate to `chrome://extensions`.
2.  Enable **Developer mode** (toggle in the top right).
3.  Click **Load unpacked**.
4.  Select the `dist-chrome` folder generated by the build command.

### Mozilla Firefox

1.  Open Firefox and navigate to `about:debugging#/runtime/this-firefox`.
2.  Click **Load Temporary Add-on...**.
3.  Navigate to the `dist-firefox` folder and select the `manifest.json` file.

## Features

- **Character Highlighting**: Automatically detects and highlights character names on supported novel sites.
- **Tooltips**: Hover over highlights to see character descriptions, aliases, and images.
- **Context Menu**: Select any text on a page, right-click, and choose "Add Character" to quickly add them to your database.
- **Auto-Backup**: Configure daily or weekly automatic backups of your character database to your local Downloads folder.
- **Dark Mode UI**: Clean, dark-themed popup and modals.
